Welcome to ESV3+

A tool to estimate your software project


SIGNUP

What and why ESV3+?

Software projects' estimation has been a problem for many years now and we believe that more work on standarizing this field is needed. Maybe due to the lack of knowledge in this topic or maybe because maths are too hard, formal estimation methods seem to be weirdly applied.
Maths and project management theories are providing us with different techniques to solve this problem, based on risks, based on complexity, based on use cases, etc ... but as those methods take too much time in this quick moving reality, we usually apply the experts judgements when it comes to give extimates for a software project.
This app tries to help you to have an estimate that is (hopefully) more accurate than the one you can give by just giving it based on a quick look.
How does this app work?

1. Maths

After some questions, the app will calculate the project's durantion based on three formal methods. You don't have to worry about the math, but if you want to do some research, you should know that we are using the "Three Values Method", the "Enhaced Three Values Method" and "Use Case Points Method".

2. The methods

Because at first we usually need to estimate a software project with little amount of information on the table, and these methods allow to estimate without going too in depth in the project.

3. Pick one or pick them all This app allows you to estimate your project using one method or all of them at the same time, so the estimation is as accurate as possible. Just answer the questions you'll be asked and click "Estimate"!
